# Introduction
With this repository, we aim to gather all reference data collections (i.e. bone pins and 3D imaging) related to the whole shoulder girdle kinematics.
We will update the repo once the work will be submitted to a journal. But this work has already been presented at:
- SOFAMEA 2024, Nantes, France
- ISG 2024, Toronto, Canada

Upcoming talks:
- SB 2024, Compiègne, France
